# 🎮 PlayFix – Game Console Servicing Platform

**PlayFix** is a full-stack MERN web application where users can offer, manage, and book servicing for gaming consoles like PlayStation 4 and PlayStation 5. Built with modern web technologies, it includes robust authentication, role-based dashboard features, booking workflows, and service status updates.

## ⚙️ Technologies Used

- **React.js** – Frontend library
- **MongoDB** – NoSQL database
- **Express.js** – Backend framework
- **Node.js** – Server runtime
- **Tailwind CSS** – Utility-first CSS framework
- **Firebase Auth** – Authentication (Email/Password + Google)
- **JWT** – JSON Web Token for secure API access
- **Framer Motion / AOS** – Animations
- **React Router** – Client-side routing
- **SweetAlert2** – Elegant alerts and notifications
- **Vite** – Build tool for frontend

---

## ✨ Features

- 🔐 **Secure Authentication**
  - Email/Password login
  - Google OAuth login
- 📋 **Service Management**
  - Add, Update, and Delete PS4/PS5 servicing offers
  - View all services and single service detail
- 📅 **Service Booking**
  - Book services from other users
  - Form with auto-filled booking details
- 🗂 **Dashboard Features**
  - 📦 Booked Services: List of services you booked
  - 🛠️ Services To Do: List of your services booked by others with status control (Pending ➜ Working ➜ Completed)
- 💡 **Search & Filter**
  - Search for services by name
- 🌗 **Theme Toggle**
  - Switch between light and dark themes
- 📱 **Fully Responsive**
  - Optimized for desktop, tablet, and mobile
- ⚙️ **JWT Protected APIs**
  - Backend routes secured with JWT token
- 📢 **Dynamic Page Titles**
  - Title updates based on current route
- 🔄 **Persistent Login**
  - User remains logged in on page reload
- ⚠️ **Custom 404 Page**
  - With redirect to home button
